# Epic ID 文字チェッカー

Epic ID / Fortnite 表示名向けの特殊文字チェッカーです。

## Files

- `index.html`: Webツール本体
- `.nojekyll`: GitHub Pages の Jekyll 処理を無効化する空ファイル
- `LICENSE`: ライセンス表記
- `README.md`: この説明ファイル

## GitHub Pages 公開方法

1. GitHubで新しいリポジトリを作成します。
2. このzip内のファイルをすべてリポジトリ直下にアップロードします。
3. `Settings` → `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. Branchを `main`、フォルダを `/root` にして保存します。

公開URLは通常、次の形式になります。

```text
https://<GitHubユーザー名>.github.io/<リポジトリ名>/
```
